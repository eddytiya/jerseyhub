# 📖 About The Project

JerseyHub is a full-stack football jersey e-commerce application that allows users to browse jerseys from clubs and international teams, securely purchase products, manage their orders, and track deliveries.

The application also provides a powerful Admin Dashboard to manage products, categories, orders, users, inventory, reviews, notifications, and analytics.

The project was developed to demonstrate real-world MERN Stack development practices including authentication, payment integration, cloud storage, responsive UI, REST APIs, and database management.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/eddytiya/JerseyHub.git
```

```bash
cd JerseyHub
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

## Backend

```env
MONGO_URI=

SESSION_SECRET=

EMAIL_USER=
EMAIL_PASS=

GOOGLE_CLIENT_ID=

RAZORPAY_KEY_ID=
RAZORPAY_KEY_SECRET=

FRONTEND_URL=

NODE_ENV=
```

## Frontend

```env
VITE_API_URL=

VITE_GOOGLE_CLIENT_ID=

VITE_CLOUDINARY_CLOUD_NAME=
VITE_CLOUDINARY_UPLOAD_PRESET=

VITE_RAZORPAY_KEY=
```
